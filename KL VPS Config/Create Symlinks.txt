In VPS01:
SymLinksIfOwnerMatch: This is a stricter security option. 
	If enabled, the web server will only follow a symbolic link if the symlink and the target file or directory have the same owner. 
	If the symlink is owned by root and the target directory is owned by kapal45, this would block access.
